# Monster Tamer

A 2D monster-taming RPG built with Unity, inspired by classic creature-collection games.  
**Explore, battle, and collect creatures in a fully data-driven, state-driven system.**

🎮 [Play Demo / Watch Gameplay](https://your-portfolio-site.com)

<img width="100%" alt="MonsterTamerBattle" src="https://github.com/user-attachments/assets/00b7f8cf-36fe-450a-87ff-1c66cd1fca4f" />

---

## ⚡ Features

- Overworld exploration with NPC interaction  
- Wild monster encounter system  
- Turn-based battle system  
- Trainer battle encounters  
- Battle flow implemented using the **State Pattern**  
- Party management system (swap, reorder, and manage monsters)  
- Inventory and item usage system  
- Monster summary and stats interface  
- View Manager / UI navigation system  
- Dialogue system  
- **Data-driven monster architecture using ScriptableObjects**

---

## 🛠 Project Highlights

- Fully **ScriptableObject-driven monster data**, allowing for easy additions and balancing  
- **State Pattern battle system**, managing transitions cleanly between idle, attack, damage, and battle end  
- Modular **UI / View Manager**, enabling flexible menus, party management, and inventory navigation  
- Scalable **party and inventory systems**, supporting swapping, reordering, and item usage

---

## 🏗 Built With

- **Engine:** Unity 2022.3.50f1 (LTS)  
- **Scripting:** C# / .NET  
- **Version Control:** Git / GitHub Desktop
