# MonsterTamer
A 2D monster-taming RPG built with Unity, inspired by classic creature-collection games.
This project focuses on clean architecture, scalability, and maintainability.
Prioritizes a robust architectural foundation for long-term development.
# Build With
- Engine: Unity 2022.3.50f1 (LTS)
- Scripting: C# / .NET
- Version Control: Git / GitHub Desktop
